To set up the repo to work, do the following:
1. Have python and pip installed.
2. run `python -m venv .venv` in your terminal.
3. run `./.venv/bin/activate` on Mac/Linux OS, or `.\.venv\Scripts\Activate.ps1` on Windows OS.
4. run `pip install -U -r requirements.txt` to import all necessary libraries.
